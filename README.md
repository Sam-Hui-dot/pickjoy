# PickJoy

PickJoy 是一个给选择困难、聚会抽签、奖励惩罚和日常小决定用的快乐选择器。

## Features

- 转盘、翻牌、骰子三种抽取方式
- 内置吃什么、谁来、奖励、惩罚、学习、周末模板
- 自定义选项和权重
- 抽过先移出本轮的去重模式
- 最近结果历史
- 本地保存当前抽奖盘
- 复制分享链接，让别人打开同一个盘

## Run locally

直接打开 `index.html` 即可运行。也可以用任意静态服务器预览：

```powershell
python -m http.server 5173
```

Then open `http://localhost:5173`.
