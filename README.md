# PickJoy

PickJoy 是一个给选择困难、聚会抽签、奖励惩罚和日常小决定用的快乐选择器。

## Features

- 转盘模式：可读文字、权重随机、完整转动后给结果
- 盲盒模式：选项先隐藏，高亮乱跳后揭晓一个结果
- 快抽一次：不进复杂动画，直接做一个短滚动抽取
- 内置吃什么、去哪、谁来、奖励、惩罚、学习模板
- 自定义选项和权重
- 抽过先移出本轮的去重模式
- 最近结果历史
- 本地保存当前抽奖盘
- 复制分享链接，让别人打开同一个盘

## Run locally

直接打开 `index.html` 即可运行。也可以用任意静态服务器预览：

```powershell
python -m http.server 5173
```

Then open `http://localhost:5173`.
